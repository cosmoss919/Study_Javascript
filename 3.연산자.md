# 문자열 연산자
### x == y

값이 같다.

### x === y

값이 같고 자료형도 같다.

### x != y

값이 다르다.

### x !== y

값이 다르고 자료형도 다르다.

<br/>

💥==, != 보단 ===, !== 사용을 지향함
``` javascript
// ==, ===
console.log(1 == 1, '1' == 1);
console.log(1 === 1, '1' === 1);

// !=, !==
console.log(1 != 1, '1' != 1);
console.log(1 !== 1, '1' !== 1);
```

<br/>

💥 문자열은 다른 자료형과 더해지면 문자열로 반환
``` javascript
const str = 123 + 'Hi' + true + null + undefined;
console.log(str);
```
